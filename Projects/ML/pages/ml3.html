What does it do?

This image classification app takes an image, breaks down the image data, and, using a trained convolutional neural network, classifies the image with a percent certainty. The machine learning algorithms were made using the ml5 and tensorflow javascript libraries and the canvas was drawn using the p5 javascript library.
